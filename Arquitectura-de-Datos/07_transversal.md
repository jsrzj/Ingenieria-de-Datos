# 🛡️ Capa Transversal: Gobernanza, Seguridad y Observabilidad

## 🎯 Objetivo

Garantizar que los datos sean seguros, confiables, trazables y cumplan con las políticas y normativas de la organización durante todo su ciclo de vida.

## ⚙️ Funciones

- Gestionar permisos y accesos a los datos.
- Proteger información sensible.
- Mantener catálogos y metadatos.
- Monitorizar procesos y rendimiento.
- Garantizar cumplimiento normativo.
- Facilitar auditorías y trazabilidad.

## 📝 Ejemplo

Una empresa necesita controlar quién puede acceder a los datos financieros y personales de sus clientes.

Para ello:

- Microsoft Purview mantiene el catálogo de datos.
- Azure Key Vault protege credenciales y secretos.
- Azure Monitor supervisa el estado de los servicios.
- Microsoft Defender for Cloud detecta amenazas de seguridad.

De esta forma, la organización garantiza que los datos sean seguros, estén controlados y puedan ser auditados.

## 🔑 Idea clave

La capa transversal actúa sobre toda la arquitectura de datos, asegurando la **seguridad, gobernanza y monitorización** de todos los procesos y recursos.
