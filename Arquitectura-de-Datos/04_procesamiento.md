# ⚙️ Capa de Procesamiento

## 🎯 Objetivo
Transformar los datos almacenados en la capa RAW para convertirlos en información limpia, consistente y preparada para el análisis y la toma de decisiones.

## ⚙️ Funciones
- Limpiar datos erróneos o incompletos.
- Eliminar duplicados.
- Validar la calidad de los datos.
- Aplicar reglas de negocio.
- Integrar datos de diferentes fuentes.
- Generar nuevos atributos y métricas.

## 📝 Ejemplo

Una empresa tiene datos de ventas almacenados en la capa RAW:

- Algunos registros tienen campos vacíos.
- Existen clientes duplicados.
- Las fechas tienen formatos diferentes.

Mediante Apache Spark o Azure Databricks se limpian los datos, se eliminan duplicados, se unifican formatos y se calculan métricas como ventas totales y clientes activos.

## 🔑 Idea clave

La capa de procesamiento es la **fábrica de datos**, donde la información bruta se transforma en datos confiables y útiles para el negocio.
