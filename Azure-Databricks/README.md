# Azure Databricks

## Introducción

Azure Databricks es una plataforma de análisis de datos basada en Apache Spark que permite procesar grandes volúmenes de información y desarrollar soluciones de Inteligencia Artificial y Machine Learning.

---

## ¿Qué es Azure Databricks?

Azure Databricks es un servicio cloud de Microsoft Azure diseñado para Big Data, análisis avanzado y Machine Learning.

### Características principales

- Procesamiento distribuido de datos.
- Basado en Apache Spark.
- Integración con Azure.
- Soporte para Python, SQL, Scala y R.
- Entorno colaborativo mediante notebooks.
